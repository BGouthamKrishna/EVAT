Electric Vehicle Adoption Tools (EVAT)

Empowering Australia's EV journey through AI, Machine Learning, and Smart Infrastructure Planning

Project Overview

EVAT is an interdisciplinary data science and software initiative focused on enabling seamless EV adoption in Australia. From predictive infrastructure planning to real-time user support via mobile applications, our platform merges machine learning, geospatial analysis, and user experience design into a cohesive ecosystem for smarter mobility.
